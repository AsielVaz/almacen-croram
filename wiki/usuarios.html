Administración de usuarios

Este módulo sirve para crear accesos, actualizar datos y activar o desactivar usuarios del sistema.

Datos básicos del usuario

  • Nombre
  • Usuario
  • Correo
  • Contraseña
  • Rol
  • Estado activo o inactivo

Estados posibles

  • Activo: puede entrar al sistema.
  • Inactivo: no puede iniciar sesión.

Cómo crear un usuario

  1. Abre Administrar usuarios.
  2. Selecciona Agregar usuario.
  3. Captura nombre, usuario, correo y contraseña.
  4. Asigna rol si aplica.
  5. Guarda el registro.

Cuándo desactivar o activar

  • Desactiva usuarios que ya no deben entrar al sistema.
  • Activa nuevamente un usuario si debe recuperar acceso.
  • Antes de desactivar, verifica que no tenga tareas pendientes de captura.
