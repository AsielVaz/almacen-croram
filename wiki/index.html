Cómo usar el sistema de almacén

Esta wiki explica el uso diario del sistema: acceso, registro de artículos, captura de órdenes, consulta de inventario, reportes y administración de usuarios. Está pensada para personal operativo y administrativo, no para desarrollo técnico.

Artículos Inventario Órdenes Reportes Usuarios

Ruta recomendada para empezar

  1. Inicia sesión con tu correo y contraseña.
  2. Consulta el panel inicial para ver el estado general del almacén.
  3. Verifica artículos, proveedores y órdenes según tu tarea.
  4. Usa reportes para revisar existencias, movimientos y faltantes.

Módulos principales

  • Artículos: catálogo general e inventario actual.
  • Familias y subfamilias: clasificación de artículos.
  • Proveedores: datos para compras.
  • Órdenes: entradas y salidas del almacén.
  • Reportes: consulta y exportación de información.
  • Usuarios: control de accesos al sistema.
Consejo: si no encuentras un artículo, primero usa el buscador por nombre, descripción o SKU antes de darlo de alta de nuevo.
